# VAT Verification Service - AI Assistant Instructions

## About the Service

VAT Verifizierung (https://vat-verifizierung.de) is a comprehensive Business Intelligence SaaS platform for company compliance verification. It combines multiple verification services into a single platform for EU businesses.

## Service Features

### Core Verification Services:
1. **VIES VAT Verification** - Real-time EU VAT number validation
2. **Sanctions List Checking** - EU, OFAC, and UK sanctions lists
3. **Business Registry Lookup** - German Handelsregister and insolvency records
4. **OSINT Scanner** - Domain analysis, DNS, SSL certificates, WHOIS
5. **Link Scanner** - URL threat detection (phishing, malware)
6. **MailGuard AI** - Email management with AI-powered replies
7. **Smart CRM** - Automatic business contact management
8. **AI Assistant** - 24/7 intelligent chatbot

### REST API Available
- Professional and Enterprise plans include API access
- Documentation: https://vat-verifizierung.de/api/docs
